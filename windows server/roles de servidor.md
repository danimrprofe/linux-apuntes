# Roles

* Combinación predefinida de varios servicios
* Componentes más pequeños llamados características
* WS puede ejecutar tantos roles como el hardware permita
* La recomendación es que cada servidor ejecute solo 1 o 2

Se agregan roles y características

* Gráficamente mediante un asistente en la consola de administración de servidor
* Desde la consola de PowerShell
